- 搜索结果只显示 title，界面更简洁

## 🚀 立即测试（3 步）

### 1️⃣ 重新加载插件
```
设置 → 社区插件 → Tag Click Search
关闭 → 等待1秒 → 打开
```

### 2️⃣ 创建测试内容
在任意笔记中输入：
```
#凡人修仙传  #仙逆
```

- 点击 `#凡人修仙传` → 应该只搜索 `凡人修仙传` ✅
- 点击 `#仙逆` → 应该只搜索 `仙逆` ✅
