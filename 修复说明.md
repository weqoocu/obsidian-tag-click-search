## 🐛 历史修复

### 问题 1: Null 值错误（v1.0.1 已修复 ✅）
```
Cannot read properties of null (reading 'toString')
```
**原因：** frontmatter 中的 tags 数组包含 null 值  
**修复：** 添加 null 检查

### 问题 2: 标签识别错误（v1.0.2 已修复 ✅）
**现象：** 点击 `#凡人修仙传` 会识别为 `#凡人修仙传#仙逆`  
**原因：** 收集标签片段时没有正确判断标签结束位置  
**修复：** 添加标签结束判断逻辑
