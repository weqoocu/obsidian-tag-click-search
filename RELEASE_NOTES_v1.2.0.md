# 🎉 Tag Click Search v1.2.0 - 智能标签建议功能

## ✨ 重大更新

### 🎯 智能标签建议
现在当你在搜索框中输入时，插件会自动显示匹配的标签建议列表，让你更快速地找到想要搜索的标签！

**核心特性：**
- 🔍 **实时匹配**：输入时自动显示匹配的标签
- 📊 **智能排序**：按字母顺序显示，支持中文排序
- ⚡ **快速选择**：点击建议项即可搜索
- ⌨️ **键盘导航**：支持方向键选择和回车确认
- 🎨 **美观界面**：专业的下拉建议设计

## 🎮 使用方法

### 基础使用
1. 打开标签搜索面板
2. 在输入框中开始输入标签名
3. 查看自动显示的建议列表
4. 点击建议项或按回车键搜索

### 键盘快捷操作
- **↑/↓** - 上下选择建议项
- **Enter** - 确认选择并搜索
- **ESC** - 关闭建议列表
- **直接输入并 Enter** - 搜索输入的标签

### 📸 功能演示

```
输入: "项目"
建议显示:
  # 项目管理
  # 项目/开发
  # 项目/设计
  # 个人项目
  
点击任意建议 → 自动搜索该标签
```

## 🆕 新功能详解

### 1. 自动标签收集
- ✅ 自动收集 vault 中所有已使用的标签
- ✅ 支持内容标签（`#标签`）
- ✅ 支持 frontmatter 标签
- ✅ 支持嵌套标签（`#父标签/子标签`）

- 📊 排序优化：按字母顺序智能排序
- 🎯 结果限制：最多显示 10 个最相关建议

### 3. 交互优化
- 🖱️ 鼠标悬停高亮
- ⌨️ 键盘完整导航
- 👆 点击建议自动搜索
- 🚪 点击外部自动关闭

## 📦 安装/升级方法

### 升级现有版本
如果你已经安装了 v1.1.0：
1. 下载新版本的 `main.js` 和 `manifest.json`
2. 替换原有文件
3. 重启 Obsidian

### 全新安装
1. 下载下方的 `main.js` 和 `manifest.json` 文件
2. 在你的 Obsidian vault 中创建文件夹：`.obsidian/plugins/tag-click-search/`
3. 将下载的文件放入该文件夹
4. 重启 Obsidian
5. 在 设置 → 社区插件 中启用 "Tag Click Search"

### 通过 BRAT 更新
1. 在 BRAT 插件中点击更新
2. 重启 Obsidian
